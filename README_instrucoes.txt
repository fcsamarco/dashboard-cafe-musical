
# Dashboard Financeiro - Café Musical

## Como executar localmente:

1. Certifique-se de que o arquivo `itens.xlsx` está na mesma pasta que este app.
2. Instale os pacotes necessários:
   ```
   pip install -r requirements.txt
   ```
3. Rode o app com:
   ```
   python app.py
   ```
4. Abra o navegador em `http://127.0.0.1:8050`

## Conteúdo:
- Gráfico de Compras, Custos e Recebimentos por mês.
- Atualização dinâmica por ano.
