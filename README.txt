
# Dashboard Profissional - Café Musical

## O que contém:
- Layout com Bootstrap responsivo
- Cartões de resumo com valores em R$
- Gráficos de barras e linha por mês
- Visual agradável e moderno

## Como usar:

1. Coloque `itens.xlsx` (arquivo de dados) na mesma pasta.
2. Instale os pacotes:
```
pip install -r requirements.txt
```
3. Rode:
```
python app.py
```
4. Acesse no navegador: http://127.0.0.1:8050
